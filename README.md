💰 **기획 배경**

    👩🏻 타겟 사용자
        경제적 어려움을 타인과 공유하며 유쾌하게 대처하고 싶고,
        이를 통해 소소한 실속을 챙기고 싶은 20·30세대

💰 **핵심 기능**

- 계좌 연동을 통한 자동 가계부 제공
- 성취감을 느끼고, 서로 공감할 수 있는 그룹 미션
- 미션을 통해 절약한 금액을 적금으로 저축
